# Team6_ArcadeSim

A [libGDX](https://libgdx.com/) project generated with [gdx-liftoff](https://github.com/libgdx/gdx-liftoff).

## Platforms

- `core`: shared engine/runtime logic.
- `lwjgl3`: desktop launcher and demo scenes.

## Gradle

Use the included Gradle wrapper:

- Windows: `gradlew.bat`
- macOS/Linux: `./gradlew`

Useful tasks:

- `build`: build all modules
- `test`: run tests (when present)
- `lwjgl3:run`: run desktop app
- `lwjgl3:jar`: build runnable JAR

Note that most tasks can be scoped to a module with `name:` prefix.
Example: `core:clean` cleans only the `core` module.
